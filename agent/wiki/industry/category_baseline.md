# Wiki: 类目健康度基线（女装 V1）

> 抖音电商女装类目健康度指标基线。诊断时以此为对照线，偏离 ≥ 20% 进入异常池。
> **数据来源**：mock 数据，按 2026 年 Q1-Q2 公开渠道（巨量算数 / 行业报告 / 抖店罗盘公开口径）估算还原。
> **更新频率**：每季度更新一次；规则大变更随时插入修订。
> **维护人**：贾丽婼 + 后续运营同学。

---

## 1. 女装·自播店 基线

| 指标 | 健康基线 | 异常阈值（偏离） |
|---|---|---|
| 直播间人均停留时长 | ≥ 60 秒 | < 50 秒 |
| 转粉率 | ≥ 3% | < 2% |
| UV 价值 | ≥ 8 元 | < 5 元 |
| 上链接 CTR | ≥ 15% | < 10% |
| CVR | ≥ 2.4% | < 1.9% |
| 主图 CTR | ≥ 7.5% | < 6% |
| 千川 ROI | ≥ 1.8 | < 1.0 |
| 退款率 | ≤ 7.5% | > 9% |
| 体验分 | ≥ 4.6 | < 4.3 |
