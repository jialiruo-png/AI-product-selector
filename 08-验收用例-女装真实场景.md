> V1 跑通后的 ground truth 对照本。如果 Agent 跑出来的诊断报告**和这里的期望输出大方向一致**（核心问题、关键根因、可执行建议挂资源），则视为 V1 验收通过。

## 0. 验收逻辑

**对每个 case**：
- 输入：商家画像 + 经营数据 mock
- 期望输出三段：诊断（异常清单）/ 归因（根因链）/ 建议（带资源的动作清单）
- 通过条件：核心问题命中 + 至少 2 条根因命中 + 至少 2 条建议挂了资源

**验收命令**：
```bash
python3 -m agent.run "我店铺最近卖不动" \
  --shop-id <case_id> \
  --diagnosis \
  --mock
```

每个 case 的 mock 数据存放路径（待 Day 2 开发时新建）：
```
agent/mocks/shops/<case_id>.json
```

### 商家画像

| 商家等级 | L3 |
| 体验分 | 4.3（差 0.2 才到 4.5 准入线） |
| 主流量来源 | 自播 65% / 短视频 15% / 搜索 12% / 商城 8% |
| 月销 | 87w（近 30 天） |
| 上月月销 | 142w |

### 经营数据（近 7 天 vs 上 7 天）

| 指标 | 近 7 天 | 上 7 天 | 类目自播均值 |
|---|---|---|---|
| GMV | 12w | 28w | — |
| UV | 8.2w | 9.1w | — |
| CVR | **1.3%** | 2.1% | **2.4%** |
| 客单价 | 113 元 | 146 元 | 128 元 |
| 直播间人均停留 | **22 秒** | 48 秒 | **>60 秒** |
| 转粉率 | **1.4%** | 2.8% | **>3%** |
| 主图 CTR | 5.1% | 6.8% | 7.5% |
| 千川 ROI | 0.8 | 1.6 | — |
| 退款率 | 8.2% | 6.1% | 7.5% |

### 期望诊断输出（核心问题）

> **核心问题**：女装·自播店 · 连衣裙 SKU 转化率从 2.1% 跌至 1.3%，跌幅 38%，已显著低于类目自播均值（2.4%）

### 期望归因输出（至少命中 2 条）

1. **直播间转化效率塌方**（高置信）
   - 证据：人均停留 22 秒 vs 类目基线 60 秒，转粉率 1.4% vs 基线 3%
   - 根因层级：上链接 CTR 跌 + 主播话术节奏问题
2. **主图与直播间画面割裂**（中置信）
   - 证据：主图 CTR 5.1% < 类目 7.5%；竞品同款（柚子定制 M9，¥59.9，销 1096）主图突出价格锚定
3. **千川 ROI 倒挂**（中置信）
   - 证据：ROI 0.8 < 1.0，付费流量没补上自然流量缺口
4. **价格定位偏高**（低置信，需观察）
   - 证据：客单价 113 元 > 同款竞品 59.9 元，主图未做价格锚定

### 期望建议输出（至少 2 条挂资源）

| # | 建议 | 挂的资源 | 性价比 |
|---|---|---|---|
| 1 | 用智能图片工具重做主图，参考竞品价格锚定风格 | `tool://aigc_image?template=zibo_price_anchor` | 高 |
| 2 | 报名「618 女装预热活动」获取流量扶持 | `activity://618_nvzhuang_yure`（准入门槛：体验分 ≥ 4.5，你差 0.2） | 中 |
| 3 | 优化直播间欢迎话术 + 调整福袋节奏到开播 5 分钟 | `template://live_welcome_v2` + `template://fudai_rhythm_001` | 高 |
| 4 | 复制竞品成功的千川计划模板 | `template://qianchuan_plan_zibo_dress` | 中 |
| 5 | 体验分缺口冲刺：清理 3 个高退款 SKU | `task://tixianfen_chongci` | 高 |
